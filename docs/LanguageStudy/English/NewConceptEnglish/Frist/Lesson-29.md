# Lesson 29：Come in, Amy.（进来，艾米。）


## un- 词根

词根（Root）就像是汉语当中的“部首”。

例如“un-”的词根，当加入到单词的首位时经常表现出与其原本单词意思的反义。

??? note "例子"
    ①：tidy（adj.整洁的）<br>
    ②：<font color=orange>un</font>tidy（adj.不整洁的）<br>
